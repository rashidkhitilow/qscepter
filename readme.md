## Installation

-clone the repo and cd into it
-composer install
-rename or copy .env.example file to .env
-Enter your database credentials in your .env file
-php artisan migrate
-php artisan key:generate
-npm install
-npm run dev
-php artisan serve
-Visit localhost:8000 in your browser
-db file: qscepter.sql
